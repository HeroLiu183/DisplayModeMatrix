# DisplayModeMatrix

提供單一維度 ASP.NET MVC Display Modes 下的複式組合可能性

## 多重可選維度

Device (optional) : Mobile | Tablet | Default (empty suffix)

Theme (optional) : Dark | Default (empty suffix)

Preview (optional) : Preview | No Preview (empty suffix)

## 組合及順序

- Mobile-Dark-Preview
- Tablet-Dark-Preview
- Mobile-Dark
- Tablet-Dark
- Mobile-Preview
- Tablet-Preview
- Dark-Preview
- Dark-Preview
- Dark
- Preview
